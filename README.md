AKA Selvbetjeningsløsninger
========================================

## Setup:
- Clone to local machine:
  - `git clone git@git.magenta.dk:gronlandsprojekter/aka-selvbetjening.git`
  - `cd aka-selvbetjening`
- Start docker containers:
  - `docker-compose up`
